pointers for block

  1. Width of a block level element is 100% of the view port width
  2. Height of a block isn't set by default
  3. Height and width can be explicitly set using CSS

pointers for inline

  1. Width of an inline level element is dependent on the content
  2. Height of a block isn't set by default
  3. Height and width can not be explicitly set using CSS
